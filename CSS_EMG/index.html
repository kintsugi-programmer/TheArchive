Responsive Websites : Responsive Design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes.
Relative Units css cheatsheet :
  1. em : Relative to the font-size of the element (2em means 2 times the size of the current font)
  2. rem : Relative to font-size of the root element
  3. vw : Relative to 1% of the width of the viewport
  4. vh : Relative to 1% of the height of the viewport
  5. % : Relative to the parent element
  6. px : Relative to the screen pixel
Position property
  1. static : Default value , element is positioned according to the normal flow of the document
  2. relative : Element is positioned relative to its normal position
  3. absolute : Element is positioned relative to its first positioned (not static) ancestor element
  4. fixed : Element is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
  5. sticky : Element is positioned based on the user's scroll position
Semantic Tags
  1. header : Defines a header for a document or a section
    1. nav : Defines navigation links
  2. main : Specifies the main content of a document
    1. section : Defines a section in a document
      1. article : Defines an article
    2. aside : Defines content aside from the content (like a sidebar)
  3. footer : Defines a footer for a document or a section
Semenatic tag vs Non Semenatic tag
Semenatic : Clearly describes its meaning to both the browser and the developer ,enhances SEO and SCREEN READERs
Non Semenatic : Does not clearly describes its content to the browser and the developer
